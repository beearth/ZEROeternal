# 0ETERNAL System Context & Philosophy

## 1. Brand Essence: 0 to Eternal
- **Concept**: 인간의 무지, 아픔, 결핍(0)을 탐지하여 영구적인 지식 자산(Eternal)으로 치환하는 시스템.
- **Core Value**: 도파민 기반의 단기적 소모를 지양하고, 세로토닌 기반의 지속 가능한 축적(Stack) 경제를 지향함.

## 2. Key Services & Definitions
- **0ETERNAL (Agent)**: LLM 기반의 감정/지능 해결사. 대화 중 사용자의 결핍을 'Red Signal(붉은 점)'으로 포착함.
- **WEstack (System)**: 개인의 지식과 감정을 캡슐화하여 1000년 이상 보존하는 디지털 유산 금고.
- **Red ROOM**: 탐지된 'Red Signal' 데이터들이 임시 보관되고 해결을 기다리는 공간.
- **Co-Thrive (상생순환)**: 기술이 인간을 소외시키지 않고, 모든 이해관계자가 따뜻함을 나누는 생태계 철학.

## 3. Special UX (SUX) Logic
- **Visual Marker**: 단어 우측 중앙에 배치된 붉은 점은 '미해결된 0'을 상징함.
- **Action**: 레드 시그널 클릭 시 지식 습득/문제 해결 프로세스 작동 -> 해결 시 점이 소멸되며 'Eternal Stack'으로 자산화됨.
- **Integration**: 모든 서비스의 활동은 통합 포인트로 전환되어 월 20불의 구독료를 상쇄함.
